INLINE TAGS

SPAN

The tag is an inline container used to mark up a part of a text, or a part of a document. The tag is easily styled by CSS

CITE

The tag defines the title of a creative work (e.g. a book, a poem, a song, a movie, a painting, a sculpture, etc.). Note: A person's name is not the title of a work. The text in the element usually renders in italic.

https://www.w3schools.com/tags/tag_cite.asp

CODE

The code tag is used to define a piece of computer code. The content inside is displayed in the browser's default monospace font.
